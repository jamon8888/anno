# S3 Notes (Local)

This is a project-specific note about a personal S3 bucket used for caching datasets/models and
storing evaluation artifacts. It is **not required** to use `anno` and may drift.

## Bucket

| Property | Value |
|----------|-------|
| Name | `arc-anno-data` |
| Region | us-east-1 |
| Access | Private (owner-only) |
| Created | 2024-12-08 |

### Security configuration (example)

```json
{
  "BlockPublicAcls": true,
  "IgnorePublicAcls": true,
  "BlockPublicPolicy": true,
  "RestrictPublicBuckets": true
}
```

## Layout (proposed)

```
s3://arc-anno-data/
│
├── datasets/                    # Evaluation and training datasets
├── models/                      # Pre-trained model weights
├── scripts/                     # Utility scripts
├── eval-results/                # Evaluation outputs (date-organized)
├── experiments/                 # Experimental artifacts
└── manifests/                   # Checksums and provenance
```

See the old, more detailed structure notes in git history if needed.

## Environment (example)

Add to `.env`:

```bash
# S3 cache configuration
ANNO_S3_CACHE=1
ANNO_S3_BUCKET=arc-anno-data

# HuggingFace token (for gated datasets)
HF_TOKEN=hf_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## Sync examples

```bash
# List bucket contents
s5cmd ls s3://arc-anno-data/

# Upload a model directory
s5cmd sync ~/.cache/huggingface/hub/models--knowledgator--gliner-x-small/ \
  s3://arc-anno-data/models/ner/gliner/gliner-x-small/
```
