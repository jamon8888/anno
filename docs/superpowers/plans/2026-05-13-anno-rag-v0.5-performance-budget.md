# anno-rag v0.5 Performance Budget — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land 4 memory/latency optimizations + a CI-gated bench harness so anno-rag stays within a hard performance budget (peak RSS <1.5 GB, cold-start <2s, search p95 <200ms) on an 8 GB / 4 cores laptop.

**Architecture:** Each optimization lands as an independently mergeable task. T1 (bench harness) lands FIRST so every subsequent task is measured. Lazy-init (T2), OCR drop (T3), and fp16 (T5) are independent. T4 (single-backend NER) depends on v0.3 #001 being merged. T6 wires the user-facing `anno-rag bench` CLI and refreshes the bench doc with v0.5 numbers.

**Tech Stack:** Rust 2021, criterion 0.5 for benches, sysinfo 0.32 for cross-platform RSS, candle 0.10 (already pinned) for F16 dtype, tokio::sync::OnceCell for lazy init, tokio::process::Command for OCR fork.

**Spec:** `docs/superpowers/specs/2026-05-13-anno-rag-v0.5-performance-budget.md`

```rust
//! Shared helpers for benches.
#![allow(dead_code)] // each bench uses a subset

use anno_rag::{AnnoRagConfig, Pipeline};
use sysinfo::{Pid, ProcessRefreshKind, RefreshKind, System};

/// Path to the bench fixture corpus relative to crate root.
pub fn bench_corpus_dir() -> std::path::PathBuf {
    std::path::PathBuf::from(env!("CARGO_MANIFEST_DIR"))
        .join("tests/fixtures/bench_corpus")
}

/// Build a Pipeline pointing at a fresh temp data dir. Caller owns the TempDir.
pub async fn pipeline_in_tempdir() -> (Pipeline, tempfile::TempDir) {
    let tmp = tempfile::tempdir().expect("tempdir");
    let mut cfg = AnnoRagConfig::default();
    cfg.data_dir = tmp.path().to_path_buf();
    let key = [0u8; 32];
    let p = Pipeline::new(cfg, key).await.expect("pipeline");
    (p, tmp)
}

/// Current process RSS in bytes, cross-platform. Returns 0 on lookup failure.
pub fn current_rss_bytes() -> u64 {
    let mut sys = System::new_with_specifics(
        RefreshKind::new().with_processes(ProcessRefreshKind::new().with_memory()),
    );
    let pid = Pid::from_u32(std::process::id());
    sys.refresh_process(pid);
    sys.process(pid).map(|p| p.memory()).unwrap_or(0)
}
```

//! Measures RSS after Pipeline::new but before any tool call. Hard gate: <200 MB.

mod common;

use criterion::{criterion_group, criterion_main, Criterion};
use tokio::runtime::Runtime;

fn bench_idle_rss(c: &mut Criterion) {
    c.bench_function("idle_rss_mb", |b| {
        b.to_async(Runtime::new().unwrap()).iter_custom(|iters| async move {
            let start = std::time::Instant::now();
            for _ in 0..iters {
                let (p, _tmp) = common::pipeline_in_tempdir().await;
                let rss = common::current_rss_bytes();
                // Emit the measurement to stdout so CI scraping can read it.
                eprintln!("IDLE_RSS_BYTES={}", rss);
                drop(p);
            }
            start.elapsed()
        });
    });
}

criterion_group!(benches, bench_idle_rss);
criterion_main!(benches);
```

//! Measures recall@10 across 10 reference queries. Hard gate: ≥ 95% of main baseline.
//! Emits RECALL_AT_10=<float> on stderr for CI scraping.

mod common;

use criterion::{criterion_group, criterion_main, Criterion};
use tokio::runtime::Runtime;

const REFERENCE_QUERIES: &[(&str, &str)] = &[
    ("résiliation contrat avec préavis",     "doc1.md"),
    ("conseil juridique fourniture",         "doc1.md"),
    ("identifiant numéro de sécurité",       "doc2.md"),
    ("coordonnées bancaires IBAN",           "doc2.md"),
    ("clause confidentialité informations",  "doc3.md"),
    ("non concurrence employé",              "doc3.md"),
    ("modalités de paiement facture",        "doc4.md"),
    ("indemnité licenciement",               "doc4.md"),
    ("droit applicable juridiction",         "doc5.md"),
    ("résolution amiable litige",            "doc5.md"),
];

fn bench_recall(c: &mut Criterion) {
    let rt = Runtime::new().unwrap();
    let (pipeline, _tmp) = rt.block_on(async {
        let (p, tmp) = common::pipeline_in_tempdir().await;
        p.ingest_folder(
            &common::bench_corpus_dir(),
            true,
            &tmp.path().join("outputs"),
        )
        .await
        .expect("ingest");
        (p, tmp)
    });

    c.bench_function("recall_at_10", |b| {
        b.to_async(&rt).iter(|| async {
            let mut hits = 0_usize;
            for (q, expected) in REFERENCE_QUERIES {
                let res = pipeline.search(q, 10).await.unwrap();
                if res.iter().any(|h| h.source_path.ends_with(expected)) {
                    hits += 1;
                }
            }
            let recall = hits as f64 / REFERENCE_QUERIES.len() as f64;
            eprintln!("RECALL_AT_10={}", recall);
        });
    });
}

criterion_group!(benches, bench_recall);
criterion_main!(benches);
```

```rust
use std::sync::Arc;
use tokio::sync::OnceCell;

pub struct Pipeline {
    embedder: OnceCell<Arc<Embedder>>,
    detector: OnceCell<Arc<Detector>>,
    store: Store,
    vault: Vault,
    cfg: AnnoRagConfig,
}

impl Pipeline {
    pub async fn new(cfg: AnnoRagConfig, vault_key: [u8; 32]) -> Result<Self> {
        std::fs::create_dir_all(&cfg.data_dir).map_err(Error::from)?;
        let vault = Vault::open(&cfg.vault_path(), vault_key)?;
        let store = Store::open(&cfg).await?;
        Ok(Self {
            embedder: OnceCell::new(),
            detector: OnceCell::new(),
            store,
            vault,
            cfg,
        })
    }

    async fn embedder(&self) -> Result<&Arc<Embedder>> {
        self.embedder
            .get_or_try_init(|| async {
                Embedder::load(&self.cfg).await.map(Arc::new)
            })
            .await
    }

    fn detector_get_or_init(&self) -> Result<&Arc<Detector>> {
        // Detector::new is sync today. Use set_if_not_set pattern via blocking init.
        if let Some(d) = self.detector.get() {
            return Ok(d);
        }
        let d = Arc::new(Detector::new()?);
        let _ = self.detector.set(d);
        Ok(self.detector.get().expect("just set"))
    }

    #[must_use]
    pub fn embedder_loaded(&self) -> bool {
        self.embedder.initialized()
    }

    #[must_use]
    pub fn detector_loaded(&self) -> bool {
        self.detector.initialized()
    }

    // ... ingest_one / search / detect / etc. — replace direct field reads
    // (self.embedder, self.detector) with self.embedder().await? and
    // self.detector_get_or_init()?
}
```

### Step 3.3 — Write the ocr module

```rust
//! System tesseract fork for PDFs without a text layer.
//!
//! Off by default. Enable via `--enable-ocr` (or `enable_ocr = true` in TOML).
//! Tesseract must be installed: `winget install tesseract-ocr.tesseract-ocr`,
//! `brew install tesseract`, or `apt install tesseract-ocr tesseract-ocr-fra`.

use crate::error::{Error, Result};
use std::path::{Path, PathBuf};
use tokio::process::Command;

/// Run system tesseract on `pdf_path`, returning the extracted plain text.
/// `tesseract_path` is the binary path (defaults to "tesseract" on PATH).
///
/// # Errors
/// Returns [`Error::Detect`] if the binary is missing, the fork fails, or the
/// process exits non-zero. Timeout 60s.
pub async fn ocr_pdf(pdf_path: &Path, tesseract_path: Option<&PathBuf>) -> Result<String> {
    let bin = tesseract_path
        .map(|p| p.as_os_str().to_owned())
        .unwrap_or_else(|| std::ffi::OsString::from("tesseract"));

    let out = tokio::time::timeout(
        std::time::Duration::from_secs(60),
        Command::new(&bin)
            .arg(pdf_path)
            .arg("-") // stdout
            .arg("-l")
            .arg("fra+eng")
            .output(),
    )
    .await
    .map_err(|_| Error::Detect("tesseract: 60s timeout".into()))?
    .map_err(|e| Error::Detect(format!("tesseract fork: {e}")))?;

    if !out.status.success() {
        let stderr = String::from_utf8_lossy(&out.stderr);
        return Err(Error::Detect(format!(
            "tesseract exited {}: {}",
            out.status, stderr
        )));
    }
    String::from_utf8(out.stdout).map_err(|e| Error::Detect(format!("tesseract output utf8: {e}")))
}
```

```rust
//! `anno-rag bench --corpus <dir>` — reproduce SLO measurements on a user corpus.

use crate::error::Result;
use crate::{AnnoRagConfig, Pipeline};
use std::path::Path;
use std::time::Instant;

const QUERIES: &[&str] = &[
    "résiliation contrat",
    "clause confidentialité",
    "préavis de fin",
    "objet du contrat",
    "modalités de paiement",
];

pub async fn run(corpus: &Path) -> Result<()> {
    let tmp = tempfile::tempdir().map_err(crate::error::Error::from)?;
    let mut cfg = AnnoRagConfig::default();
    cfg.data_dir = tmp.path().to_path_buf();
    let key = [0u8; 32];

    // Cold-start
    let t0 = Instant::now();
    let p = Pipeline::new(cfg, key).await?;
    let cold = t0.elapsed();

    // Idle RSS
    let rss_idle = current_rss_mb();

    // Ingest
    let t1 = Instant::now();
    let n = p.ingest_folder(corpus, true, &tmp.path().join("out")).await?;
    let ingest = t1.elapsed();
    let rss_after_ingest = current_rss_mb();

    // Search latencies
    let mut search_lat_us = Vec::new();
    for q in QUERIES {
        let t = Instant::now();
        let _ = p.search(q, 10).await?;
        search_lat_us.push(t.elapsed().as_micros());
    }
    search_lat_us.sort_unstable();
    let p50 = search_lat_us[search_lat_us.len() / 2];
    let p95 = search_lat_us[search_lat_us.len() * 95 / 100];
    let rss_peak = current_rss_mb();

    println!("# anno-rag bench results\n");
    println!("| Metric | Value | SLO |");
    println!("|---|---|---|");
    println!("| Cold-start | {:.2}s | <2s |", cold.as_secs_f64());
    println!("| Idle RSS | {} MB | <200 MB |", rss_idle);
    println!("| Ingest ({} docs) | {:.1}s ({:.1}s/doc) | <10s/doc |",
             n, ingest.as_secs_f64(), ingest.as_secs_f64() / n as f64);
    println!("| Search p50 | {} µs | — |", p50);
    println!("| Search p95 | {} µs | <200000 µs |", p95);
    println!("| Peak RSS | {} MB | <1500 MB |", rss_peak);
    println!("| RSS after ingest | {} MB | — |", rss_after_ingest);
    Ok(())
}

fn current_rss_mb() -> u64 {
    use sysinfo::{Pid, ProcessRefreshKind, RefreshKind, System};
    let mut sys = System::new_with_specifics(
        RefreshKind::new().with_processes(ProcessRefreshKind::new().with_memory()),
    );
    let pid = Pid::from_u32(std::process::id());
    sys.refresh_process(pid);
    sys.process(pid).map(|p| p.memory() / 1024 / 1024).unwrap_or(0)
}
```
