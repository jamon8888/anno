**Tech Stack:** rmcp 1.6 (already in workspace deps), tokio, the existing anno-rag v0.1 code.

**Prerequisites:** v0.1 merged (PR #1 on jamon8888/anno).

//! Pre-download both the embedder AND the anno NER model used by `anno-rag`.
//!
//! Run once before integration tests or the first CLI use:
//!   cargo run --example warmup_model -p anno-rag --release
//!
//! Total cache: ~600 MiB (e5-small embedder 448 MiB + gliner_pii NER ~120 MiB).

use anno_rag::config::AnnoRagConfig;
use hf_hub::api::tokio::Api;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let cfg = AnnoRagConfig::default();
    let api = Api::new()?;

    // ---- 1. Embedder ----
    println!("Warming embedder: {}", cfg.embed_model);
    let embedder_repo = api.model(cfg.embed_model.clone());
    for file in ["config.json", "tokenizer.json"] {
        let path = embedder_repo.get(file).await?;
        let size = std::fs::metadata(&path)?.len();
        println!("  {file} ... ok ({:.1} KiB) -> {}", size as f64 / 1024.0, path.display());
    }
    let weights_path = match embedder_repo.get("model.safetensors").await {
        Ok(p) => p,
        Err(_) => embedder_repo
            .get("pytorch_model.bin")
            .await
            .map_err(|e| anyhow::anyhow!("weights: {e}"))?,
    };
    let size_mb = std::fs::metadata(&weights_path)?.len() as f64 / 1024.0 / 1024.0;
    println!("  weights ... ok ({size_mb:.1} MiB) -> {}", weights_path.display());

    // ---- 2. NER (try multiple candidate backends) ----
    let ner_candidates = [
        cfg.ner_warmup_model.as_deref().unwrap_or("knowledgator/gliner-pii-edge-v1.0"),
        "onnx-community/gliner_small-v2.1",
        "numind/NuNER_Zero",
        "protectai/bert-base-NER-onnx",
    ];

    println!("\nWarming anno NER (first available backend):");
    let mut warmed_any = false;
    for model_id in ner_candidates {
        match warm_ner_model(&api, model_id).await {
            Ok(total_mb) => {
                println!("  {model_id} ... ok ({total_mb:.1} MiB)");
                warmed_any = true;
                break; // First success is enough
            }
            Err(e) => {
                println!("  {model_id} ... skip ({e})");
            }
        }
    }
    if !warmed_any {
        eprintln!(
            "\nWARN: no NER model could be downloaded. Name detection will \
             fall back to pattern+heuristic and miss many French names. \
             Check network / HF Hub credentials and retry."
        );
    }

    println!("\nDone. Run integration tests with:");
    println!("  cargo test -p anno-rag --test e2e -- --ignored");
    Ok(())
}

async fn warm_ner_model(api: &Api, model_id: &str) -> anyhow::Result<f64> {
    let repo = api.model(model_id.to_string());
    let mut total: u64 = 0;
    for file in [
        "config.json",
        "tokenizer.json",
        "model.onnx",
        "model.safetensors",
        "pytorch_model.bin",
    ] {
        if let Ok(path) = repo.get(file).await {
            total += std::fs::metadata(&path)?.len();
        }
    }
    if total == 0 {
        anyhow::bail!("no candidate files downloaded");
    }
    Ok(total as f64 / 1024.0 / 1024.0)
}
```

```rust
    /// Build an `IVF_HNSW_SQ` index on the vector column if:
    ///   1. The table currently has >= `threshold` rows
    ///   2. The vector column doesn't already have an index
    ///
    /// Idempotent — calling twice does nothing the second time. Runs the
    /// index build (which is CPU-heavy) on the calling task; callers should
    /// invoke this from `tokio::spawn_blocking` or accept the latency.
    pub async fn maybe_build_index(&self, threshold: usize) -> Result<bool> {
        use lancedb::index::Index;
        use lancedb::index::vector::IvfHnswSqIndexBuilder;

        let count = self
            .tbl
            .count_rows(None)
            .await
            .map_err(|e| Error::Store(format!("count_rows: {e}")))? as usize;
        if count < threshold {
            return Ok(false);
        }

        // Skip if an index already exists on the vector column.
        let existing = self
            .tbl
            .list_indices()
            .await
            .map_err(|e| Error::Store(format!("list_indices: {e}")))?;
        if existing.iter().any(|i| i.columns.iter().any(|c| c == "vector")) {
            return Ok(false);
        }

        self.tbl
            .create_index(&["vector"], Index::IvfHnswSq(IvfHnswSqIndexBuilder::default()))
            .execute()
            .await
            .map_err(|e| Error::Store(format!("create_index: {e}")))?;
        Ok(true)
    }
```

//! MCP server exposing anno-rag's retrieval surface to Cowork.
//!
//! Tools: `search`, `rehydrate`, `detect`, `vault_stats`.
//! Transport: stdio (Cowork plugin convention).

use crate::config::AnnoRagConfig;
use crate::error::Result;
use crate::pipeline::Pipeline;
use rmcp::{
    handler::server::{router::tool::ToolRouter, wrapper::Parameters},
    model::{Implementation, ServerCapabilities, ServerInfo},
    schemars, tool, tool_handler, tool_router, ServerHandler, ServiceExt,
};
use serde::{Deserialize, Serialize};
use std::sync::Arc;

/// State held by the MCP server: a fully-initialized Pipeline.
#[derive(Clone)]
pub struct AnnoRagServer {
    pipeline: Arc<Pipeline>,
    cfg: AnnoRagConfig,
    tool_router: ToolRouter<Self>,
}

// ---- Tool parameter types ----

#[derive(Deserialize, schemars::JsonSchema)]
pub struct SearchParams {
    /// User query text. May contain PII — will be pseudonymized through the vault.
    pub query: String,
    /// Number of results to return.
    #[serde(default = "default_top_k")]
    pub top_k: usize,
}

fn default_top_k() -> usize { 10 }

#[derive(Deserialize, schemars::JsonSchema)]
pub struct RehydrateParams {
    /// Text containing pseudo-tokens (e.g. `EMAIL_1`, `PERSON_3`) to restore.
    pub text: String,
}

#[derive(Deserialize, schemars::JsonSchema)]
pub struct DetectParams {
    /// Text to scan for PII without replacement.
    pub text: String,
}

// ---- Tool response types ----

#[derive(Serialize)]
struct SearchHitWire {
    doc_id: String,
    chunk_id: String,
    source_path: String,
    folder_path: String,
    chunk_idx: u32,
    text_pseudo: String,
    page: Option<u32>,
    char_start: u32,
    char_end: u32,
    distance: f32,
}

#[derive(Serialize)]
struct SearchResult {
    hits: Vec<SearchHitWire>,
}

#[derive(Serialize)]
struct RehydrateResult {
    text: String,
    tokens_rehydrated: usize,
}

#[derive(Serialize)]
struct DetectResult {
    entities: Vec<EntityInfo>,
}

#[derive(Serialize)]
struct EntityInfo {
    original: String,
    category: String,
    confidence: f64,
    source: String,
    start: usize,
    end: usize,
}

#[derive(Serialize)]
struct VaultStatsResult {
    total_mappings: usize,
    categories: std::collections::HashMap<String, u32>,
}

// ---- Tool router ----

#[tool_router]
impl AnnoRagServer {
    #[tool(description = "Search the indexed corpus. Pseudonymizes the query through the local vault, embeds it, returns top-K ranked chunks. Chunks are pseudonymized — call rehydrate(text) to restore originals.")]
    async fn search(
        &self,
        Parameters(params): Parameters<SearchParams>,
    ) -> String {
        match self.pipeline.search(&params.query, params.top_k).await {
            Ok(hits) => {
                let wire = SearchResult {
                    hits: hits
                        .into_iter()
                        .map(|h| SearchHitWire {
                            doc_id: h.doc_id.to_string(),
                            chunk_id: h.chunk_id.to_string(),
                            source_path: h.source_path,
                            folder_path: h.folder_path,
                            chunk_idx: h.chunk_idx,
                            text_pseudo: h.text_pseudo,
                            page: h.page,
                            char_start: h.char_start,
                            char_end: h.char_end,
                            distance: h.distance,
                        })
                        .collect(),
                };
                serde_json::to_string_pretty(&wire).unwrap_or_else(|e| format!("Error: {e}"))
            }
            Err(e) => format!("Error: {e}"),
        }
    }

    #[tool(description = "Replace pseudo-tokens (PERSON_1, EMAIL_2, NIR_3, etc.) in text back with original PII from the local vault.")]
    async fn rehydrate(
        &self,
        Parameters(params): Parameters<RehydrateParams>,
    ) -> String {
        // Need to expose Vault::rehydrate via Pipeline. For v0.2 simplicity,
        // use cloakpipe_core::rehydrator directly. The pipeline owns the
        // vault; we add a Pipeline::rehydrate helper in a subsequent edit
        // — for now this method is a placeholder that returns the input
        // unchanged with tokens_rehydrated=0 if Pipeline doesn't yet
        // expose rehydrate.
        //
        // TODO before Task 6 final: add Pipeline::rehydrate(text) helper.
        let result = RehydrateResult {
            text: params.text.clone(),
            tokens_rehydrated: 0,
        };
        let _ = &self.pipeline; // suppress unused warning
        serde_json::to_string_pretty(&result).unwrap_or_else(|e| format!("Error: {e}"))
    }

    #[tool(description = "Dry-run scan: detect PII in text without replacing. Returns category, source, confidence, and char offsets for each entity.")]
    async fn detect(
        &self,
        Parameters(params): Parameters<DetectParams>,
    ) -> String {
        // Add Pipeline::detect helper or expose detector directly.
        // Placeholder for now.
        let _ = &params;
        let result = DetectResult { entities: vec![] };
        serde_json::to_string_pretty(&result).unwrap_or_else(|e| format!("Error: {e}"))
    }

    #[tool(description = "Vault statistics: total token mappings and per-category counts.")]
    async fn vault_stats(&self) -> String {
        // Add Pipeline::vault_stats helper.
        let result = VaultStatsResult {
            total_mappings: 0,
            categories: std::collections::HashMap::new(),
        };
        serde_json::to_string_pretty(&result).unwrap_or_else(|e| format!("Error: {e}"))

}

#[tool_handler]
impl ServerHandler for AnnoRagServer {
    fn get_info(&self) -> ServerInfo {
        ServerInfo::new(ServerCapabilities::builder().enable_tools().build())
            .with_instructions(
                "anno-rag MCP server. Tools: search (pseudonymized retrieval), \
                 rehydrate (restore originals), detect (dry-run scan), vault_stats.",
            )
            .with_server_info(Implementation::new(
                self.cfg.mcp_server_name.clone(),
                env!("CARGO_PKG_VERSION"),
            ))
    }
}

impl AnnoRagServer {
    pub fn new(pipeline: Pipeline, cfg: AnnoRagConfig) -> Self {
        Self {
            pipeline: Arc::new(pipeline),
            cfg,
            tool_router: Self::tool_router(),
        }
    }
}

/// Start the MCP server on stdio. Blocks until stdin closes.
pub async fn serve_stdio(pipeline: Pipeline, cfg: AnnoRagConfig) -> Result<()> {
    let server = AnnoRagServer::new(pipeline, cfg);
    tracing::info!("anno-rag MCP server starting on stdio");

    let transport = rmcp::transport::stdio();
    let service = server
        .serve(transport)
        .await
        .map_err(|e| crate::error::Error::Detect(format!("MCP server failed to start: {e}")))?;
    service
        .waiting()
        .await
        .map_err(|e| crate::error::Error::Detect(format!("MCP server error: {e}")))?;
    Ok(())
}

    /// Rehydrate pseudo-tokens in `text` back to originals using the vault.
    pub async fn rehydrate(&self, text: &str) -> Result<RehydratedText> {
        use cloakpipe_core::rehydrator::Rehydrator;
        // The cloakpipe Rehydrator needs `&Vault` but our Vault wraps it in
        // Arc<Mutex<>>. Lock for the duration of the rehydrate call.
        let guard = self.vault_inner_for_rehydrate().await;
        Rehydrator::rehydrate(text, &*guard)
            .map(|r| RehydratedText { text: r.text, tokens_rehydrated: r.rehydrated_count })
            .map_err(|e| Error::Vault(format!("rehydrator: {e}")))
    }

    /// Detect PII in `text` without replacing.
    pub fn detect(&self, text: &str) -> Result<Vec<cloakpipe_core::DetectedEntity>> {
        self.detector.detect(text)
    }

    /// Vault statistics (count + per-category).
    pub async fn vault_stats(&self) -> VaultStats {
        let guard = self.vault_inner_for_rehydrate().await;
        let s = guard.stats();

# anno-rag

Local, GDPR-compliant document anonymizer + RAG service for French legal documents.

## What it does

Ingest a folder of legal documents → detect French PII (names, NIR, SIRET, IBAN-FR, phone) → pseudonymize through an AES-256-GCM vault → index in LanceDB → expose to Claude Cowork via MCP for retrieval. All local, no cloud.

## Quick start

```sh
# Build
cargo build --release -p anno-rag

# 1. Warm the model cache (~600 MiB download, one-shot)
cargo run --release --example warmup_model -p anno-rag

# 2. Ingest a folder of French legal documents
./target/release/anno-rag ingest ~/cabinet/dossier-acme --recursive

# 3. Query from the CLI
./target/release/anno-rag search "résiliation pour cause"

# 4. Or hand off to Cowork via MCP
./target/release/anno-rag mcp
```

## Cowork plugin configuration

Add to your Cowork plugins config:

```json
{
  "anno-rag": {
    "command": "/absolute/path/to/anno-rag",
    "args": ["mcp"],
    "env": {
      "ANNO_RAG_VAULT_PASSPHRASE": "your-passphrase-here"
    }
  }
}
```

Cowork connects to anno-rag over stdio and gets these tools:

- `search(query, top_k)` — pseudonymized top-K chunks (the data Claude sees)
- `rehydrate(text)` — restore originals from the local vault (only your laptop ever has both)
- `detect(text)` — dry-run PII scan
- `vault_stats()` — token-mapping counts

## Architecture

```
ingest:  folder → kreuzberg → detect (regex + anno NER) → cloakpipe Vault
                              → embed (BGE-multilingual-e5-small) → LanceDB → outputs/*.anon.md

search:  query → pseudonymize → embed → LanceDB top-K → pseudonymized chunks
                                                          ↓
                                                       Cowork → Claude
                                                          ↓
                              (optional) Claude's answer with tokens → rehydrate → user
```

The cleartext PII lives only in the vault file on disk (`~/.anno-rag/vault.enc`). Pseudonymized copies (`outputs/*.anon.md`) and the LanceDB index never see raw PII.
