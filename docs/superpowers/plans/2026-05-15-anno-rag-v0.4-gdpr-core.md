# anno-rag v0.4 — GDPR core (rights routes + persistent audit + gateway auth) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close gaps G2–G6 of the RGPD readiness spec — implement the right-of-erasure and right-of-access primitives (`Pipeline::forget`, `Pipeline::find_subject`, export), replace `NoopAuditSink` with a tamper-evident JSONL audit register (sha256 hash chain + daily HMAC), and add bearer-token auth to the `anno-privacy-gateway` HTTP routes.

**Architecture:** Three concentric layers. (1) Vendored `cloakpipe-core::Vault` grows two new methods — `remove` and `find` — providing the primitive forget/lookup operations. (2) `anno-rag::Pipeline` exposes `forget(subject)`, `find_subject(ref)`, `export_subject(ref, format)` that compose those primitives with the audit sink. (3) `anno-privacy-gateway` adds `/v1/subjects/forget`, `/v1/subjects/find`, `/v1/subjects/{ref}/export` routes behind a bearer-token middleware, and replaces `NoopAuditSink` with a `JsonlAuditSink` writing append-only `YYYY-MM-DD.jsonl` files with per-line sha256 chaining and a daily HMAC signature file. The gateway does NOT take an `anno-rag` dependency; the routes operate on the gateway's own `PrivacyEngine`/`Vault` so the privacy-gateway crate stays standalone.

**Tech Stack:** Rust, axum 0.8, tokio, cloakpipe-core (vendored), sha2, hmac, subtle (constant-time compare), tower-http auth, serde_json. **Note for Windows dev (NMarchitecte):** all build/test commands run in WSL with `CARGO_TARGET_DIR=/tmp/anno-v0.5-target` to keep the build cache on Linux ext4. Re-create the dir if it was wiped.

```rust
/// Remove the mapping for `subject_ref`, which may be either the original
/// value or its pseudo-token. Returns the removed mapping on success, or
/// `None` if no mapping matched. Counters are NOT decremented — token
/// ids remain monotonic so future tokens never collide with retired ones.
///
/// This is the primitive used to honour GDPR Art. 17 (right to erasure):
/// after this call, neither the original nor the token resolves to a
/// vault entry; rehydration of any pseudonymised text containing the
/// retired token will leave the token in place.
pub fn remove(&mut self, subject_ref: &str) -> Option<RemovedMapping> {
    // Case 1: subject_ref is an original (look up its token via forward).
    if let Some(token) = self.forward.remove(subject_ref) {
        self.reverse.remove(&token.token);
        return Some(RemovedMapping {
            original: subject_ref.to_string(),
            token: token.token,
            category: token.category,
        });
    }
    // Case 2: subject_ref is a token (look up its original via reverse).
    if let Some(sensitive) = self.reverse.remove(subject_ref) {
        let original = sensitive.0;
        // Drop the forward entry whose token matches.
        if let Some(token) = self.forward.remove(&original) {
            return Some(RemovedMapping {
                original,
                token: token.token,
                category: token.category,
            });
        }
        // forward map missing the back-reference — should not happen, but
        // surface what we have so the audit record is complete.
        return Some(RemovedMapping {
            original,
            token: subject_ref.to_string(),
            category: EntityCategory::Custom("Unknown".into()),
        });
    }
    None
}

/// Find every vault entry matching `subject_ref` — searched against both
/// originals (forward) and tokens (reverse). Currently returns 0 or 1
/// matches (fuzzy resolver not consulted; that would be a v0.5 extension).
pub fn find(&self, subject_ref: &str) -> Vec<MatchedMapping> {
    if let Some(token) = self.forward.get(subject_ref) {
        return vec![MatchedMapping {
            original: subject_ref.to_string(),
            token: token.token.clone(),
            category: token.category.clone(),
        }];
    }
    if let Some(sensitive) = self.reverse.get(subject_ref) {
        if let Some(token) = self.forward.get(&sensitive.0) {
            return vec![MatchedMapping {
                original: sensitive.0.clone(),
                token: token.token.clone(),
                category: token.category.clone(),
            }];
        }
    }
    Vec::new()
}
```

```rust
/// Remove the vault mapping for `subject_ref` (original or token). Persists
/// the vault to disk on success. Returns the removed mapping for audit, or
/// `None` if no mapping matched.
///
/// # Errors
/// Returns [`Error::Vault`] if persisting the vault to disk fails.
pub async fn forget(&self, subject_ref: &str) -> Result<Option<RemovedMapping>> {
    let mut v = self.inner.lock().await;
    let removed = v.remove(subject_ref);
    if removed.is_some() {
        v.save()
            .map_err(|e| Error::Vault(format!("save after forget: {e}")))?;
    }
    Ok(removed)
}

/// Find every vault entry matching `subject_ref` (original or token).
pub async fn find_subject(&self, subject_ref: &str) -> Vec<MatchedMapping> {
    let v = self.inner.lock().await;
    v.find(subject_ref)
}
```

```rust
/// Receipt returned by [`Pipeline::forget`]. Suitable for inclusion in an
/// audit event.
#[derive(Debug, Clone, serde::Serialize)]
pub struct ErasureReceipt {
    /// Whatever the caller passed (original or token).
    pub subject_ref: String,
    /// Number of vault mappings removed (0 if subject was unknown).
    pub mappings_removed: usize,
    /// Token that was retired, if a mapping was found.
    pub token: Option<String>,
    /// Category of the retired mapping, if any.
    pub category: Option<String>,
    /// UTC timestamp of the operation, RFC 3339.
    pub executed_at: String,
}

/// Result of [`Pipeline::find_subject`]. Multiple matches may be returned
/// (the cloakpipe primitive currently returns 0 or 1 entries; the type is
/// vec-shaped for a future fuzzy match).
#[derive(Debug, Clone, serde::Serialize)]
pub struct SubjectMatches {
    /// The subject reference the caller looked up.
    pub subject_ref: String,
    /// Zero or more matches.
    pub matches: Vec<SubjectMatch>,
}

/// One match returned by [`Pipeline::find_subject`].
#[derive(Debug, Clone, serde::Serialize)]
pub struct SubjectMatch {
    /// Original (sensitive) value.
    pub original: String,
    /// Pseudo-token in the vault.
    pub token: String,
    /// Category key (e.g. `"Person"`, `"Email"`, `"NIR"`).
    pub category: String,
}

/// Output format for [`Pipeline::export_subject`].
#[derive(Debug, Clone, Copy)]

    /// JSON object: `{ "subject_ref": ..., "matches": [...] }`.
    Json,
    /// CSV with header: `original,token,category`.
```

```rust
/// Remove every vault mapping that matches `subject_ref` (original or
/// token). Idempotent: returns a receipt with `mappings_removed = 0` if
/// nothing matched. Persists the vault on success.
///
/// # Errors
/// Returns [`Error::Vault`] if the vault could not be persisted.
pub async fn forget(&self, subject_ref: &str) -> Result<ErasureReceipt> {
    let removed = self.vault.forget(subject_ref).await?;
    let now = chrono::Utc::now().to_rfc3339();
    Ok(match removed {
        Some(m) => ErasureReceipt {
            subject_ref: subject_ref.to_string(),
            mappings_removed: 1,
            token: Some(m.token),
            category: Some(format!("{:?}", m.category)),
            executed_at: now,
        },
        None => ErasureReceipt {
            subject_ref: subject_ref.to_string(),
            mappings_removed: 0,
            token: None,
            category: None,
            executed_at: now,
        },
    })
}

/// Look up every vault mapping for `subject_ref` (original or token).
pub async fn find_subject(&self, subject_ref: &str) -> SubjectMatches {
    let matches = self
        .vault
        .find_subject(subject_ref)
        .await
        .into_iter()
        .map(|m| SubjectMatch {
            original: m.original,
            token: m.token,
            category: format!("{:?}", m.category),
        })
        .collect();
    SubjectMatches {
        subject_ref: subject_ref.to_string(),
        matches,
    }
}

/// Export the matches for `subject_ref` in the requested format. JSON
/// returns the `SubjectMatches` shape; CSV returns
/// `original,token,category` rows with a header.
///
/// # Errors
/// Returns [`Error::Audit`] if CSV serialisation fails (extremely
/// unlikely — strings only).
pub async fn export_subject(
    &self,
    subject_ref: &str,
    format: ExportFormat,
) -> Result<Vec<u8>> {
    let res = self.find_subject(subject_ref).await;
    match format {
        ExportFormat::Json => serde_json::to_vec_pretty(&res)
            .map_err(|e| Error::Audit(format!("json serialize: {e}"))),
        ExportFormat::Csv => {
            let mut buf = Vec::new();
            {
                let mut w = csv::Writer::from_writer(&mut buf);
                w.write_record(["original", "token", "category"])
                    .map_err(|e| Error::Audit(format!("csv header: {e}")))?;
                for m in &res.matches {
                    w.write_record([&m.original, &m.token, &m.category])
                        .map_err(|e| Error::Audit(format!("csv row: {e}")))?;
                }
                w.flush()
                    .map_err(|e| Error::Audit(format!("csv flush: {e}")))?;
            }
            Ok(buf)
        }
    }
}
```

## Task 4: Persistent `JsonlAuditSink` (sha256 chain + daily HMAC)

**Files:**
- Modify: `crates/anno-privacy-gateway/src/audit.rs`
- Modify: `crates/anno-privacy-gateway/Cargo.toml` (deps: `sha2`, `hmac`, `hex`, `time`)
- Modify: `crates/anno-privacy-gateway/src/config.rs` (fields)

Replaces `NoopAuditSink` with a real Art. 30 register. **File layout** (under `config.audit_dir`, default `~/.local/state/anno-rag/audit/`):

```
audit/
  2026-05-15.jsonl   # one line per event
  2026-05-15.sig     # HMAC-SHA256 of the day's final chain head
```

Each JSONL line is:

```json
{"ts":"2026-05-15T13:42:11Z","event":{...},"prev_hash":"<hex>","this_hash":"<hex>"}
```

`prev_hash` is `0000…` for the first line of the day; `this_hash = sha256(prev_hash || serde_json::to_vec(event))`. The `.sig` file is written/rewritten on every event and contains `HMAC-SHA256(hmac_key, this_hash)` for the *last* event of the day — a verifier can replay the chain and compare.

```rust
/// Safe request audit metadata. Never includes cleartext prompts, rehydrated
/// answers, or vault contents.
#[derive(Debug, Clone, serde::Serialize, serde::Deserialize, PartialEq, Eq)]
pub struct AuditEvent {
    pub request_id: String,
    pub provider_profile: String,
    pub entity_count: usize,
    pub fresh_pii_redacted: usize,
}

/// Audit sink abstraction.
pub trait AuditSink: Send + Sync {
    fn record(&self, event: AuditEvent);
}

/// No-op sink — for tests only.
#[derive(Debug, Default)]
pub struct NoopAuditSink;
impl AuditSink for NoopAuditSink {
    fn record(&self, _event: AuditEvent) {}

/// Persistent tamper-evident JSONL sink.
///
/// Layout under `dir`:
/// - `YYYY-MM-DD.jsonl` — one event per line, with `prev_hash` / `this_hash`
/// - `YYYY-MM-DD.sig` — HMAC-SHA256(key, last_this_hash) in hex
///
/// Chain head is `0` × 64 hex chars at the start of each UTC day, then
/// `this_hash = sha256(prev_hash_bytes || serde_json::to_vec(event))`.
pub struct JsonlAuditSink {
    dir: PathBuf,
    hmac_key: [u8; 32],
    state: Mutex<DailyState>,
}

struct DailyState {
    /// `YYYY-MM-DD` of the file currently being appended.
    day: String,
    /// Hex of the most recent `this_hash` for the current day.
    chain_head: String,
}

impl JsonlAuditSink {
    /// Initialise the sink. If the dir doesn't exist it's created. If
    /// today's JSONL already has lines, the chain head is recovered from
    /// the last line so chaining survives restarts.
    pub fn new(dir: impl Into<PathBuf>, hmac_key: [u8; 32]) -> std::io::Result<Self> {
        let dir = dir.into();
        std::fs::create_dir_all(&dir)?;
        let day = current_day();
        let chain_head = read_last_chain_head(&dir, &day)?;
        Ok(Self {
            dir,
            hmac_key,
            state: Mutex::new(DailyState { day, chain_head }),
        })
    }

    fn write_line(&self, event: &AuditEvent) -> std::io::Result<()> {
        let today = current_day();
        let mut state = self
            .state
            .lock()
            .map_err(|_| std::io::Error::other("audit mutex poisoned"))?;
        if state.day != today {
            // Day rolled over — start a new chain.
            state.day = today.clone();
            state.chain_head = "0".repeat(64);
        }
        let prev = state.chain_head.clone();
        let event_bytes = serde_json::to_vec(event)
            .map_err(|e| std::io::Error::other(format!("serialize: {e}")))?;
        let mut hasher = Sha256::new();
        hasher.update(hex::decode(&prev).unwrap_or_default());
        hasher.update(&event_bytes);
        let this_hash = hex::encode(hasher.finalize());

        let line = serde_json::json!({
            "ts": OffsetDateTime::now_utc()
                .format(&time::format_description::well_known::Rfc3339)
                .unwrap_or_default(),
            "event": event,
            "prev_hash": prev,
            "this_hash": this_hash,
        });
        let mut serialised = serde_json::to_vec(&line)
            .map_err(|e| std::io::Error::other(format!("serialize line: {e}")))?;
        serialised.push(b'\n');

        let jsonl_path = self.dir.join(format!("{}.jsonl", state.day));
        let mut f = OpenOptions::new()
            .create(true)
            .append(true)
            .open(&jsonl_path)?;
        f.write_all(&serialised)?;
        f.sync_data()?;

        // Update sig file.
        let mut mac = HmacSha256::new_from_slice(&self.hmac_key)
            .map_err(|e| std::io::Error::other(format!("hmac init: {e}")))?;
        mac.update(this_hash.as_bytes());
        let sig = hex::encode(mac.finalize().into_bytes());
        let sig_path = self.dir.join(format!("{}.sig", state.day));
        std::fs::write(&sig_path, sig)?;

        state.chain_head = this_hash;
        Ok(())
    }
}

impl AuditSink for JsonlAuditSink {
    fn record(&self, event: AuditEvent) {
        // Audit must NEVER panic the request path. Log + drop on error.
        if let Err(e) = self.write_line(&event) {
            tracing::error!("audit write failed: {e}");
        }
    }
}

fn current_day() -> String {
    OffsetDateTime::now_utc()
        .format(&format_description!("[year]-[month]-[day]"))
        .expect("static format")
}

fn read_last_chain_head(dir: &Path, day: &str) -> std::io::Result<String> {
    let path = dir.join(format!("{day}.jsonl"));
    let f = match OpenOptions::new().read(true).open(&path) {
        Ok(f) => f,
        Err(e) if e.kind() == std::io::ErrorKind::NotFound => {
            return Ok("0".repeat(64));
        }
        Err(e) => return Err(e),
    };
    let mut last: Option<String> = None;
    for line in BufReader::new(f).lines() {
        let line = line?;
        if let Ok(v) = serde_json::from_str::<serde_json::Value>(&line) {
            if let Some(h) = v.get("this_hash").and_then(|x| x.as_str()) {
                last = Some(h.to_string());
            }
        }
    }
    Ok(last.unwrap_or_else(|| "0".repeat(64)))
}
```

```rust
/// Reject requests without a valid `Authorization: Bearer <token>` header.
/// `/health` is exempt — operational checks should not require credentials.
pub async fn require_bearer(
    State(state): State<AppState>,
    req: Request<Body>,
    next: Next,
) -> Result<Response, StatusCode> {
    if req.uri().path() == "/health" {
        return Ok(next.run(req).await);
    }
    let configured = state
        .bearer_token()
        .ok_or(StatusCode::INTERNAL_SERVER_ERROR)?;
    let header = req
        .headers()
        .get(axum::http::header::AUTHORIZATION)

        .ok_or(StatusCode::UNAUTHORIZED)?;
    let eq: bool = provided.as_bytes().ct_eq(configured.as_bytes()).into();
    if !eq {
        return Err(StatusCode::UNAUTHORIZED);
    }
    Ok(next.run(req).await)
}
```

```rust
//! End-to-end GDPR test: pseudonymise → forget → audit chain verifies.

use anno_privacy_gateway::audit::{AuditSink, JsonlAuditSink};
use anno_privacy_gateway::server::{router, AppState};
use anno_privacy_gateway::GatewayConfig;
use axum::body::Body;
use axum::http::Request;
use sha2::{Digest, Sha256};
use tower::ServiceExt;

fn cfg(audit_dir: &std::path::Path, hmac_key_hex: &str) -> GatewayConfig {
    let mut c = GatewayConfig::default();
    c.bearer_token = Some("secret".into());
    c.audit_dir = Some(audit_dir.to_path_buf());
    c.audit_hmac_key_hex = Some(hmac_key_hex.to_string());
    c
}

#[tokio::test]
async fn forget_persists_to_audit_chain() {
    let tmp = tempfile::TempDir::new().unwrap();
    let key_hex = hex::encode([7u8; 32]);
    let app = router(AppState::try_new(cfg(tmp.path(), &key_hex)).unwrap());

    for i in 0..3 {
        let req = Request::builder()
            .method("POST")
            .uri("/v1/subjects/forget")
            .header("authorization", "Bearer secret")
            .header("content-type", "application/json")
            .body(Body::from(format!(r#"{{"subject_ref":"s{i}"}}"#)))
            .unwrap();
        let res = app.clone().oneshot(req).await.unwrap();
        assert_eq!(res.status(), 200);
    }

    let today = time::OffsetDateTime::now_utc()
        .format(&time::macros::format_description!("[year]-[month]-[day]"))
        .unwrap();
    let body = std::fs::read_to_string(tmp.path().join(format!("{today}.jsonl"))).unwrap();
    let lines: Vec<&str> = body.lines().collect();
    assert_eq!(lines.len(), 3);

    let mut prev = "0".repeat(64);
    for line in &lines {
        let v: serde_json::Value = serde_json::from_str(line).unwrap();
        assert_eq!(v["prev_hash"].as_str().unwrap(), prev);
        let event_bytes = serde_json::to_vec(&v["event"]).unwrap();
        let mut h = Sha256::new();
        h.update(hex::decode(&prev).unwrap());
        h.update(&event_bytes);
        let expected = hex::encode(h.finalize());
        assert_eq!(v["this_hash"].as_str().unwrap(), expected);
        prev = expected;
    }
}
```
